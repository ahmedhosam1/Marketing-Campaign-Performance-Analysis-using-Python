 Marketing Data Analysis Project

📊 Overview
This project focuses on analyzing marketing campaign data to extract valuable insights about customer behavior, campaign performance, and conversion rates. The goal is to help improve marketing strategies using data-driven decisions.

---

🎯 Objectives
- Understand customer demographics and behavior
- Analyze campaign performance across different channels
- Identify factors affecting conversion rate
- Extract actionable insights to improve marketing ROI

---

📁 Dataset Description
The dataset contains information about marketing campaigns, including:
- Customer demographics (Age, Gender, Income)
- Campaign details (Channel, Type, Ad Spend)
- Engagement metrics (Click Through Rate, Website Visits, Time on Site)
- Conversion metrics (Conversion Rate, Previous Purchases)

---

🛠️ Tools & Technologies
- Python 
- Pandas
- NumPy
- Matplotlib / Seaborn
- Jupyter Notebook

---

🔍 Key Analysis Performed
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Visualization of customer behavior
- Campaign performance comparison
- Correlation analysis between features

---

📈 Key Insights
- Which campaign channel performs best
- Relationship between income and conversion rate
- Impact of ad spend on engagement
- Customer segments with highest conversion
